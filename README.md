# 📋 Attendance Management System

A simple, mobile-friendly attendance website designed for teachers to mark student attendance and quickly view absent students.

---

## 🚀 Features

* ✔ Mark students as Present/Absent
* 🔍 Search students instantly
* ⚡ Mark all Present / Absent with one click
* 📱 Fully mobile-friendly
* 📊 Displays list of absent students

---

## 🔒 Security

* Right-click disabled
* Developer tools shortcuts disabled
* Code obfuscated to prevent easy editing

> Note: This provides basic protection. Advanced users may still access code.

---

## 🧑‍🏫 How to Use

1. Open the website link
2. Search or scroll to find student names
3. Tick checkbox for Present students
4. Click **“Show Absent Students”**
5. View the absent list below

---

## 📂 Project Structure

* `index.html` → Main application file

---

## ⚙️ How to Deploy (GitHub Pages)

1. Create a repository on GitHub
2. Upload your file and rename it to `index.html`
3. Go to **Settings → Pages**
4. Select:

   * Source: `Deploy from branch`
   * Branch: `main`
5. Save and wait 1–2 minutes
6. Your site will be live 🎉
